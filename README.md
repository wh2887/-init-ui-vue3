# Init UI

一款基于 Vue 3 技术栈开发的极简 UI 组件库。

手机扫码预览
![手机扫码](https://gitee.com/hellow2887/blogimage/raw/master/img/init-qrCode.png)

## 技术栈
- Vue 3
- TypeScript
- Vue Router 4
- Vite
- SCSS 

## 组件
- Button 按钮
- Dialog 弹出对话框
- Switch 开关切换
- Tabs 标签页

## 安装依赖

```bash
yarn install
```

## 开发

```bash
yarn dev
```

## 构建

```bash
yarn build
```
